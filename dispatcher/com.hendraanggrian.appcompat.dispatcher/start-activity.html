startActivity

fun <T : Activity> T.startActivity(intent: Intent, callback: T.(resultCode: Int, data: Intent?) -> Unit): Unit

Start an activity for result with auto-generated request code. The result will then have to be handled in callback instead of Activity.onActivityResult.

Parameters

intent - the intent to start.

callback - activity result callback.

Exceptions

ActivityNotFoundException - if no activity is found that can handle the intent.
